# Project Euler Solutions - Go

100 [Project Euler](https://projecteuler.net/) solutions in Go, part of a cross-language benchmark suite with standardized benchmarking across 11 languages.

See [ProjectEuler.Benchmarks](https://github.com/august-hill/ProjectEuler.Benchmarks) for cross-language performance comparisons.

## Prerequisites

- **Runtime**: `go` (1.21+)
- **Platform**: macOS / Linux

## Running

```bash
# Run a single solution
cd problem_001
go run .

# Run all benchmarks (from repo root)
./run_all.sh
```

## Project Structure

```
ProjectEuler.Go/
  bench/                    # Shared benchmarking package
  problem_001/main.go       # Solution + benchmark for problem 1
  problem_002/main.go
  ...
  problem_100/main.go
```

Each `main.go` imports the `bench` package and outputs timing results in a standardized format consumed by the Benchmarks aggregator.
